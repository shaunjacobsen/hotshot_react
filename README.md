# Hotshot
This is an example implementation of various front-end web components for Adyen.
_Hotshot_ is a fake company, which is intended to act as a SaaS company that sells POS and order management software to basketball arenas.
The model is such that the retailers (food/bev and merchandise) are onboarded as "sub-merchants" and can sell goods via two channels: POS terminals or online (e-commerce).

# About
I have provided this repository primarily as a way to test and demonstrate the front-end implementation of Adyen's components for my clients.
The back-end is a private repository; Adyen's [own documentation](https://docs.adyen.com/online-payments/build-your-integration) should be used as an example for how to build the server layer.
